h2. sbt-appbundle

h3. statement

sbt-appbundle is a plugin for xsbt (sbt 0.11) that adds the @appbundle@ task to copy a project's packaged jar along with its dependencies into the @Contents/Resources/Java@ folder of an OS X application bundle.
