BosqueTimelineCursor


This class represents a (graphical user interface) cursor position in a session's timeline. At the moment, there is exactly one cursor available:


~doc    = Bosque.default.session;

~cursor = ~doc.timelineView.cursor;


timeline


Returns the BosqueTimeline instance to which this cursor refers.


position, position_( <(Integer) newPos> )


Queries or sets the cursor position in sample frames. Note however, that you should not call position_ directly, but use the editPosition method of the BosqueSession instead, since that way the positioning operation is properly inserted in the undo manager.


Modification of the cursor position results in a notification ( \changed, newPos ) being delivered to registered observers. Often you want to observe the BosqueTimelineView instead however.
