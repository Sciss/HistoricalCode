BosqueEnvSegmentStake


The env object of a BosqueEnvRegionStake is a BosqueTrail whose elements are instances of BosqueEnvSegmentStake. For computational ease, the breakpoint function / envelope is not stored as a list of points but instead as a list of segments. Each segment has a duration (specified by the stake's span), a startLevel (the value of the curve at the span's start) and a stopLevel (the value of the curve at the span's stop). For internal consistency, the startLevel of a segment should always be equal to the stopLevel of the predecessor segment.


Furthermore, a shape (type) and curve value can be specified in accordance to the SuperCollider Env object. However, as of the current version, only linear segments (shape == \lin and curve == 0.0) are supported. This may change in a future version.


startLevel


The value (Float between 0 and 1) of the segment at the span's start.



stopLevel


The value (Float between 0 and 1) of the segment at the span's stop.



shape


The segments shape (Symbol). Must be \lin at the moment.



curve


Additional curvature parameter depending on the shape of the segment. Should be 0 at the moment.


level( <(Integer) frame> )


Calculates the level (Float between 0 and 1) of the segment at a given frame position. The frame should be contained in the stake's span.
