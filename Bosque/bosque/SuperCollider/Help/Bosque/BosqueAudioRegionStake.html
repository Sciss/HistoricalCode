BosqueAudioRegionStake


A stake that represents a part of an audio file. An example of how to create and add a BosqueAudioRegionStake is found in the section about funcPreviousBus in the BosqueFuncRegionStake help file.


audioFile


Points to the corresponding BosqueAudioFile instance.



fileStartFrame


Specifies the offset (Integer frames) in the audio file corresponding to the stake span's start position.
