BosqueEnvRegionStake


A region stake that represents a breakpoint function (envelope). To add a stake to the session's trail, use the addStake method in BosqueTimelineEditor.


env


Points to a BosqueTrail containing the envelope segments. Each segment is an instance of BosqueEnvSegmentStake. The trail has a normalized origin, e.g. its span starts at zero.



level( <(Integer) frame> )


Queries the envelope value (Float between 0 and 1) at a given frame. The frame is measured against the global time base, i.e. against the trail containing the env region stake, i.e. the session timeline's span.


Usually you will prefer to query envelope values via the level and map methods of BosqueTrack.
