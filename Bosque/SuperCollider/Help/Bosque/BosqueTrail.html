BosqueTrail


A BosqueTrail is a class competing with BosqueSessionCollection. However, the trail sorts its elements (instances of Stake, e.g. BosqueRegionStake or BosqueMarkerStake objects) by time and allows you to access objects by time frames or time spans. Also, the stakes are considered immutable objects, so editing a stake is basically creating a modified copy of a stake, removing the old stake and adding the new one.


The trail manages a double-sorted list of the stakes: once sorted by their span's start value, once sorted by their span's stop value. Many methods have a byStart argument that specifies whether the operation is to be performed on the list sorted by start value (true, the default) or the one sorted by stop values (false).


span


Returns the Span covered by the trail, e.g. the Span between the start of the first stake and the stop of the last stake.



get( <(Integer) idx>, <(Boolean) byStart = true> )


Returns the Stake at a given index idx.



numStakes


Returns the number of stakes (Integer) included in the trail.



contains( <(Stake) stake> )


Queries whether a given stake is included in the trail or not (Boolean).



indexOf( <(Stake) stake>, <(Boolean) byStart = true> )


Returns the index (Integer) of a given stake in one of the two stake lists maintained by the trail. A value < 0 indicates that the stake is not included in the trail.



indexOfPos( <(Integer) pos>, <(Boolean) byStart = true> )


Returns the "insertion index" (Integer) of a given frame pos (position) in the trail. If a stake exists whose span contains the given position, its index is returned. Otherwise the returned value is (-(insertion point) - 1). The insertion point is defined as the point at which a stake with a given span's start (for byStart == true) or stop (for byStart == false) would be inserted into the trail's list: the index of the first stake whose start/stop value is greater than pos, or numStakes, if all stakes in the list have a span start/stop of less than the specified pos.



getAll( <(Boolean) byStart = true> )


Returns a copy of the internal Collection of stakes, either sorted by start or by stop.



add( <(Object) source>, <(Stake) stake> )


Inserts a stake into the trail. The source object is used to identify the originator of the modification when firing a notification.



addAll( <(Object) source>, <(Collection) stakes> )


Inserts several stakes into the trail. The source object is used to identify the originator of the modification when firing a notification.



remove( <(Object) source>, <(Stake) stake> )


Removes a stake from the trail. The source object is used to identify the originator of the modification when firing a notification.



removeAll( <(Object) source>, <(Collection) stakes> )


Removes several stakes from the trail. The source object is used to identify the originator of the modification when firing a notification.
