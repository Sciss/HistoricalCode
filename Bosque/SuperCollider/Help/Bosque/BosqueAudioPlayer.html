BosqueAudioPlayer


This class is responsible for spawning Synths on the scsynth Server during realtime playback (see BosqueTransport). The audio player is accessible from


Bosque.default.session.audioPlayer;


The audio player will manage the invocation of the playToBundle and stopToBundle methods of the session trail's BosqueRegionStake objects, such as BosqueAudioRegionStake, BosqueEnvRegionStake, and BosqueFuncRegionStake.


The audio player is passed in to the playToBundle method of a BosqueFuncRegionStake's function object, and has many useful methods for the function object. They are described in the help file for BosqueFuncRegionStake.


For an organization scheme of the Group graph, see Bosque_Audio.


As of the current version, the audio player cannot handle dynamic playback speed changes, so only works properly for a constant playback rate of 1.0.
