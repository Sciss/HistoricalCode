BosqueFade


Description of a fade in or fade out associated with a BosqueRegionStake. The BosqueAudioPlayer currently uses the fades to blend in and out BosqueAudioRegionStake objects, and it passes in the fade information to the function object of a BosqueFuncRegionStake. The meaning of the fades is currently undefined for BosqueEnvRegionStake objects.


type


The curvature type of the fade (Symbol). As of the current version, only \lin for linear fades is supported. This may change in a future version.



numFrames


The duration of the fade in sample frames.



curve


An additional curvature parameter depending on the curve type. For linear curves, this value is not used and should be 0.
