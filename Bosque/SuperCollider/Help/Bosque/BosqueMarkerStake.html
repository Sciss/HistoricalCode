BosqueMarkerStake


This class represents the markers found in a BosqueSession. They are accessed via the session's markers BosqueTrail:


~doc     = Bosque.default.session;

~markers = ~doc.markers;


pos


Returns the marker's position in sample frames (Integer).



span


Returns the marker's Span, equal to Span( pos, pos ).



name


Returns the marker's name (Symbol).



Adding markers


(Note: in this example we assume the current session has already a timeline span of some few seconds or more). The easiest way to create a marker is to use the addMarker method of the BosqueTimelineEditor:


~editor = Bosque.default.timelineEditor;

~mark   = ~editor.addMarker( ~doc, 33333, \MyMark ); // doc, position, name


Quick marker access


There is a quick marker access class method in Bosque:


~mark = Bosque.mark( \MyMark );  // using the name as the argument


Currently, marker names need not be unique, so if there are more markers with the same name, the behaviour of this method is undefined (any of the markers having the specified name may be returned).
