BosqueRegionStake


The superclass of region objects placed in the tracks of the timeline. The concrete subclasses are:


BosqueEnvRegionStake

BosqueAudioRegionStake

BosqueFuncRegionStake


The stakes of the session are accessible via the BosqueTrail


Bosque.default.session.trail


span


Returns the Span covered by the region inside the session's timeline.


name


Returns the name of region (Symbol).


track


Returns the BosqueTrack on which the stake is located.


colr


Returns the Color with which the stake is painted on the GUI.


fadeIn, fadeOut


Returns instances of BosqueFade, describing the fade in and fade out of the region.


gain


Returns a gain factor (Float) to apply when playing back the region. This is a linear value, e.g. 1.0 means no boost or attenuation.
