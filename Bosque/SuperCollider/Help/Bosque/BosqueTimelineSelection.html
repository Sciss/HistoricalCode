BosqueTimelineSelection


This class represents a (graphical user interface) selection span in a session's timeline. At the moment, there is exactly one selection available:


~doc       = Bosque.default.session;

~selection = ~doc.timelineView.selection;


timeline


Returns the BosqueTimeline instance to which this selection refers.


span, span_( <(Span) newSpan> )


Queries or sets the selection span (start and stop are measured in sample frames). Note however, that you should not call span_ directly, but use the editSelect method of the BosqueSession instead, since that way the selecting operation is properly inserted in the undo manager.


Modification of the selection span results in a notification ( \changed, newSpan ) being delivered to registered observers. Often you want to observe the BosqueTimelineView instead however.
