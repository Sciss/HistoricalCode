Bosque

statement

Bosque is a system for multitrack timeline composition with algorithmic elements in SuperCollider. It is (C)opyright 2006–2009 by Hanns Holger Rutz. All rights reserved. Bosque is released under the GNU General Public License and comes with absolutely no warranties. To contact the author, send an email to contact at sciss.de

For project status, API and current version, visit www.sciss.de/bosque.

requirements / installation

Bosque requires recent SuperCollider 3 and SwingOSC installations, and compiles against Java 1.5. The repository contains a project file for Eclipse 3.5.
